<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/Shekhar-Joshi07/Doggo.git
```

<p>2. After cloning install NPM packages</p>

```
npm i --force
```

<p>3. View on browser at localhost:3000</p>

```
npm start
```
